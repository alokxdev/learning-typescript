# Learning TypeScript

This repository contains my practice files while learning TypeScript.

## What I'm Covering

- Basic types
- Function types
- Type annotations
- Interfaces
- Implementing interfaces
- Union types
- Type intersections
- Typed arrays
- Enums
- Generics
- Type checking and compilation using `tsc`

## Project Structure

- `src/01-function-types.ts` – Function typing examples
- `src/02-interface-types.ts` – Interface and object typing examples
- `src/03-implementing-interfaces.ts` – Using interfaces in classes/objects
- `src/04-basic-type-union.ts` – Basic union type examples
- `src/05-custom-types-union.ts` – Custom union type examples
- `src/06-type-intersection.ts` – Type intersections (combining multiple types)
- `src/07-array-types.ts` – Typed arrays and array methods
- `src/08-enums.ts` – Enum examples and use cases
- `src/09-generics.ts` – Generic functions and reusable type utilities
- `tsconfig.json` – TypeScript configuration
